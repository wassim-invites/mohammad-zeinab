✦ Step-by-Step Setup
1.

Create GitHub Account

Go to github.com and sign up.

2.

Create Repository

Click + → New repository. Name it zaher-mira. Set to Public. Check "Add a README". Click Create.

3.

Upload Files

Click Add file → Upload files. Upload invite.html and admin.html. Click Commit changes.

4.

Enable GitHub Pages

Go to Settings → Pages → Source → Deploy from branch → main / (root). Click Save.

5.

Your site is live!

After 1-2 minutes:
https://YOUR-USERNAME.github.io/zaher-mira/admin.html
https://YOUR-USERNAME.github.io/zaher-mira/invite.html

6.

Create GitHub Token

Go to github.com → Settings → Developer settings → Personal access tokens → Fine-grained.
Generate token. Set: Expiration = No expiration. Repository = zaher-mira. Permissions → Contents: Read and write. Copy the token and paste it in Settings tab.

7.

Generate Guest Links

Fill in all Settings, click Save & Sync. Then go to Generate, enter guest name, copy link, send via WhatsApp.

Photos tip: Upload photos to a photos/ folder in your GitHub repo and use paths like photos/hero.jpg in the Settings fields.

✦ Cross-Device RSVP (Google Apps Script)

This makes every guest's RSVP appear in your dashboard from any device.

1.

Open Google Apps Script

Go to script.google.com → New project. Delete existing code. Paste the code below.

2.

Deploy as Web App

Click Deploy → New deployment → Web app. Execute as: Me. Who has access: Anyone. Click Deploy. Copy the URL.

3.

Paste in Settings

Go to Settings tab → Google Apps Script Webhook URL. Paste the URL. Click Save & Sync.
